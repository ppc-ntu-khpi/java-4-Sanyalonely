# Практична робота: Ієрархія класів "Леопард"

**Обрана тварина:** Леопард.

<img src="https://github.com/ppc-ntu-khpi/java-4-Sanyalonely/blob/master/images/Namibie_Etosha_Leopard_01edit.jpg" width="100%"/>

### Структура проекту:
- **Animal**: Базовий клас з загальними рисами (ім'я, вага).
- **Predator**: Клас хижаків, додає поведінку полювання (`hunt`).
- **Leopard**: Кінцевий клас, що перевизначає звуки та спосіб харчування, а також додає специфічну гру.
